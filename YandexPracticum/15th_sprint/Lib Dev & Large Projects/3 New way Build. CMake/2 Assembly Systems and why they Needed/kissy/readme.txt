1. В одной папке расоложить kissy.exe скомпилированный файл и две папки input и output
2. В папку input положить входной файл с кошкой.
3. В папке output создать выходной пустой файл.
4. В GitBush выполнить команды: 
$ ./kissy.exe ./input/1.kissy++.ppm ./output/1.kissy++_out.ppm

5. Программа должна сообщить: Image saved successfully!
