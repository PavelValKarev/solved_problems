1. Собрать .exe файл с помощью cmake
2. Получить файл negate.exe


1. В одной папке расположить negate.exe скомпилированный файл и две папки input и output
2. В папку input положить входной файл с негативным псом.
3. В папке output создать выходной пустой файл формата ppm.
4. В GitBush выполнить команды: 
$ ./negate.exe ./input/2.dog-neg.ppm ./output/2.dog-neg_out.ppm

5. Программа должна сообщить: Image saved successfully!
